Para ejecutar el programa es necesario tener instalado pocketsphinx, junto con tener descargados los archivos es-20k.lm.bin, es.dict y la carpeta es-es dentro del directorio /usr/local/lib/python2.7/dist-packages/pocketsphinx/model
Estos archivos se descargan de: https://sourceforge.net/projects/cmusphinx/files/Acoustic%20and%20Language%20Models/Spanish/

Se debe correr una simulación de Rviz para correr el programa junto con el servicio de navegación del modelo y el servicio de reproducción de voz.

El programa se puede correr con el comando de: rosrun proyecto_navigation navigation.py
